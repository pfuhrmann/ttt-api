Tic-tac-toe API
===============
API for the Tic-tac-toe game

Requirements
------------
* [PHP 7.1+](http://php.net)
* [Composer](https://getcomposer.org) 

Testing
-------
Unit tests are provided in the `/tests` folder. To run these steps simply run following command from the project root:

```bash
bin/phpunit
```
